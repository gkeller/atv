Advanced Table View

This is a advanced QTableView Class where custom widgets can be placed around the table view. For example QLineEdits in a header Row for column filters. The custom widgets dynamically resize and scroll in sync with the QTableView columns/rows. This is implemented in a single class (AdvancedTableView) which extends QTableView. At the moment only header rows are supported. The aim of the project is to implement also footer rows and side columns and improve stability, performance and plattform compatibility.

    ------------------------------------------
    |Custom Widget|...another Widget         |  <- 0...n Header Rows
    ------------------------------------------

    ------------------------------------------
    |QTableView   |                          |
    ------------------------------------------
    |             |                          |
    ------------------------------------------
